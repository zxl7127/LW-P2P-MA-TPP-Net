# LW-P2P-MA-TPP-Net
Dataset Release

This repository is dedicated to providing the datasets used in our paper:

“For Automated Cell Culture Systems: A High-Speed Style-Transfer and Prior-Guided Network for High-Precision Cell Monitoring in Bright-Field Microscopy”

📌 Availability

The datasets used in this study will be gradually released in this GitHub repository.

Due to data organization and permission considerations, the uploading process is being carried out in stages. We will update the repository continuously until all datasets are fully available.

📂 Contents

The repository will include:

Dataset-1
Dataset-2
Dataset-3
Annotation files for a subset of images in Dataset-1 (partially annotated data)

⏳ Status

🚧 Dataset upload is currently in progress.

Please check back periodically for updates.

If you have any questions or specific requests regarding the datasets, please feel free to contact us.
